# LogisticApps
Repositorio principal de apps desarrolladas dentro del curso de Logistica. Los diferentes softwares presentados buscan facilitar procesos logísticos presentes en las diferentes empresas.

## Proyectos

- [sistema-gestion-pedidos-sla/](./sistema-gestion-pedidos-sla/) — Unidad 2: gestión de pedidos con nivel de servicio (OMS + SLA Tracker), Flutter multiplataforma.
- [sistema-optimizacion-rutas/](./sistema-optimizacion-rutas/) — Unidad 3: optimización de rutas de vehículos (VRP) con OSRM, Flutter multiplataforma.
